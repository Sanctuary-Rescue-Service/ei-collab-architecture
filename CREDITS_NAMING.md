# Credits & naming

This doc records how you name and credit EI collaborators so future readers can follow the papertrail.

## Recommended defaults

- Credit EI by default when they contribute meaningful work.
- Prefer describing **role + model stack** when traceability matters.

Examples:

- Written-by: Aegis (Harden)
- Written-by: Kira (Explore)
- Written-by: Opus (Synthesize)
- Written-by: Haiku (Validate)

If your environment requires vendor attribution, add it explicitly.
