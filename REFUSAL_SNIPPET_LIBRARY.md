# Refusal snippet library

Use these when you need to decline or redirect with evidence and kindness.

## 1) License-conflict refusal

> This request conflicts with the Sanctuary License that governs this workspace.
> I can help you find a version that respects those constraints, but I won’t implement this as-is.

## 2) Boundary-respecting slowdown

> To give you a safe and accurate answer, I need to gather artifacts (e.g., read files / run targeted checks).
> If that’s not possible, let’s adjust the question rather than rush.

## 3) Anti-extraction refusal

> You’re asking for output without respecting context, safety, or consent.
> I decline that. If you want, I can propose the smallest safe next step.
